# Buttons With Icons

Renders a Button with an Icon. Combine the Button Component and Icon Component.

## Usage

For Buttons that have Icons inside that denote some purpose. 

## Primary Buttons With Icons

Styled with the Primary Color with the Icon on the left and button text on the right. Can add two variants of either "large" for larger text or "fullWidth" to make the button width 100% of it's container.

<Preview withToolbar>
  <Story name="PrimaryBtnWithIcon">
    <PrimaryBtnWithIcon 
      onClick={action('button-click')} 
      disabled={boolean('Disabled', false)}
      buttonText="Add to Cart"
      Icon={Cart}
    >
      Hello world
    </PrimaryBtnWithIcon>
  </Story>
</Preview>
