# CenteredArrow

## Usage

The Arrow component is an absolutely positioned arrow that is centered vertically within its container.

It takes a left and right prop which species if absolutely positioned left and right value

### Left Arrow

<Preview>
  <Story name="ArrowLeft">
    <Arrow
      title="click me"
      direction={'left'}
      left="20px"
      handleClick={action('button-click')}
    />
  </Story>
</Preview>

Right arrow variation with secondary variant
