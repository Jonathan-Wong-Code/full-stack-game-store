# GameReviewForm

## Usage

The game review form is a component used for reviewing games. It allows the user to set a rating of 1 to 5. They can also write a review title and description and submit it.

It is used for creating reviews and updating reviews. If it is a review type is set to update an will populate the form with the
values of the review being updated.
